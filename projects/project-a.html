프로젝트 A — 개인화 추천 시스템

사용자 행동 로그와 제품 메타데이터를 결합한 추천 모델을 설계·배포하여 전환율을 개선한 프로젝트입니다.

Python LightGBM AWS Docker

문제

기존 추천 시스템은 개인화가 부족해 CTR(클릭률)과 전환율이 낮았습니다. 사용자 세분화와 실시간 반영이 필요했습니다.

접근

행동 로그, 카테고리, 시간대 등 피처를 조합해 랭킹 모델을 학습했습니다. LightGBM 기반 offline 학습과 온라인 피처 파이프라인을 구축했으며, A/B 테스트로 모델 성능을 검증했습니다.

결과

신규 랭킹 모델 도입 후 전환율이 12% 개선되었고, 추천에 따른 평균 주문 금액이 증가했습니다. 모델은 CI/CD 파이프라인을 통해 주기적으로 재학습됩니다.
