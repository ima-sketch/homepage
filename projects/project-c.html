프로젝트 C — 이상 탐지 시스템

로그 기반 이상치 탐지 모델을 도입해 운영 이슈를 조기에 발견하고 대응 시간을 단축했습니다.

PyTorch Autoencoders Grafana

문제

운영 로그에서 이상 징후를 사람이 일일이 확인하던 방식으로는 대응이 늦었습니다. 자동화된 이상 탐지가 필요했습니다.

접근

Autoencoder 기반의 비지도 학습 모델을 적용해 정상 패턴을 학습하고 이상치를 탐지했습니다. 알림 시스템과 대시보드 연동으로 운영팀에 실시간 경고를 제공했습니다.

결과

이상 탐지 도입 후 평균 이슈 발견 시간이 단축되었고, 서비스 가용성 개선에 기여했습니다.
